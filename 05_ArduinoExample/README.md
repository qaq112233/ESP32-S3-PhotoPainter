## Steps
- Compile and flash the project to the board.
- Open the serial monitor after flashing is completed.
- Double-press the BOOT button to start recording.
- Short-press the BOOT button to play the recorded audio.
- Double-press the KEY button to play a piece of music.
- Short-press the KEY button to interrupt the music being played.
